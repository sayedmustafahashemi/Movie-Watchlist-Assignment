## ✨ Features

### Core Requirements ✅
- **Node.js + Express** server
- **MongoDB + Mongoose** database
- **RESTful API** (Full CRUD operations)
- **EJS templating engine**
- **TailwindCSS** for styling
- **Passport.js authentication** (Local Strategy)
- **Protected routes** (Authentication required)

### Additional Features 🚀
- **TMDB API Integration** - Fetch real Hindi & Hollywood movies
- **Browse Movies** - Search and discover new films
- **One-Click Add** - Add movies from browse to personal collection
- **Movie Posters** - Automatic poster images from TMDB
- **Watch Status** - Planning, Watched, Favorite
- **Stats Dashboard** - View total movies, average rating, etc.
- **Responsive Design** - Works on all devices
